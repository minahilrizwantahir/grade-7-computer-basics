What is a Computer?

A computer is an electronic machine that accepts data, processes it, and gives useful information as output. It makes our work easy and fast.

Main Parts of a Computer

  • Monitor: Shows information on the screen
  • CPU: The brain of the computer
  • Keyboard: Used to type
  • Mouse: Used to point and click

Software

Software is a set of programs that tells the computer what to do.

  • System Software: Windows, Linux
  • Application Software: MS Word, Paint, Browser

Uses of Computer

  • Education and learning
  • Typing and drawing
  • Internet and communication
  • Games and entertainment
